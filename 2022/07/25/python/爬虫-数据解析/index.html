概述

回顾requests模块实现爬虫的步骤：

  1. 指定url
  2. 发情请求
  3. 获取响应数据
  4. 持久化存储

起始在持久化存储之前还有一步数据解析，需要使用聚焦爬虫，爬取网页中部分数据，而不是整个页面的数据。下面会学习到三种数据解析的方式。至此，爬虫流程可以修改为:

  1. 指定url
  2. 发起请求
  3. 获取响应
  4. 数据解析
  5. 持久化存储

数据解析分类：

  • 正则
  • bs4
  • xpath

数据解析原理概述：

  • 解析的局部的文本内容都会在标签之间或者标签对应的属性中进行存储
  • 进行指定标签的定位
  • 标签或者标签对应的属性中存储的数据值进行提取（解析）
