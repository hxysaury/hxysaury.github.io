虚拟机安装及破解

速度：
CPU>内存>硬盘

内存可保存数据，但是断电数据消失

server2008

1、安装

选择镜像

2、启动

点击下一步，点击现在安装

选择==安全安装==，点击下一步

接受许可条款

选择自定义安装

等待安装完成

3、配置

修改密码

分辨率调整

选择适合自己电脑的分辨率

添加桌面图标

字体调整

开启远程桌面

rdp协议，端口3389

查看IP地址

远程连接

远程连接： win+r  mstsc 
输入IP地址，点击链接
输入用户名administrator
输入密码 !A123

远程连接成功

激活

#桌面点击计算机右键属性查看激活状态

#通过远程连接，拷贝激活软件,在本机里选中复制，在远程桌面里右键粘贴

#虚拟里运行神龙，不可真实机操作，防止真实机出现问题

先安装net framework 软件

断开网络

网络禁用后，双击神龙软件

再启用本地连接，打开网络

然后点击神龙激活Win Server 2008

点击确定后，退出程序，退出神龙软件时，会弹出网页，关闭即可

然后删除软件，

查看系统是否激活

关机

快照

关机后再拍摄快照

克隆

克隆类型选择==完全克隆==，如果选择链接克隆的话，再原始虚拟机删除情况下，克隆后的虚拟机就不能使用了

删除虚拟机

更改网卡

# NAT模式:如果虚拟机设置为NAT模式，该虚拟机通过主机上网，能与互联网通信

# 桥接模式:该虚拟机可以和主机分配到同一局域网的IP地址，其他主机能与该虚拟机互相通信

# 仅主机模式:该虚拟机不能和互联网通信，仅能和主机通信

1、遇到虚拟机设置NAT模式，虚拟机无法获取IP地址的情况
    关闭所有虚拟机，，，点击左上角编辑按钮--》虚拟网络编辑器   还原虚拟网络
2、无法与虚拟机通信，但是虚拟机分配到了IP地址
    （1）主机的两个虚拟网络卡没了，可以打开控制面板--》程序卸载里找到VMware，点击下一步点击修复
    （2）检查虚拟机防火墙是否关闭

4、破解密码

F10保存重启

按下任意键

diskpart
list volume 
exit

cd Windows

copy osk.exe oskbak.exe
copy cmd.exe osk.exe

win7密码破解

安装步骤和server 2008一样

在未登录系统时，连续按5次shift键，弹出程序c:\windows\system32\sethc.exe   程序

破解win7系统密码步骤：
1、开启win7虚拟机，开机，并设置一个复杂密码；
2、关机，并开机，在出现windows启动界面时强制关机；
3、再开机，出现“启动修复（推荐）”及选择该项； 
4、出现系统还原提示，点击取消，等待几分钟后，会出现问题原因，点击查看详细信息；
5、打开最后一个链接即一个记事本；
6、记事本中点打开，并选择显示所有文件；
7、找到sethc并改名sethc-bak，再找到cmd，复制一份cmd改名为sethc
8、全部关闭，重启。
9、系统启动完毕后，连续按5次shift键，将弹出cmd工具，使用命令net user 用户名 新密码，将当前用户密码修改掉即可

多次关机开机Win7直到出现==启动启动修复（推荐）==，按回车

等待一会儿弹出系统还原，点击取消，等待修复

等待。。。。。。。。。。。。

点击==查看问题详细信息==，往下滑动，滑动到最后，会出现一个.txt链接，双击打开，弹出记事本界面，

点击左上角文件，点击打开

c:\windows\system32\找到sethc.exe，进行重命名sethc-back

同样的在system32下找到cmd.exe,右键复制一份，将复制出来的重命名为sethc

点击取消按钮

关闭记事本，关闭启动修复，重启计算机

重启到登录界面后，按5次shift键，弹出cmd窗口

命令行中输入net user zs 123.com ，回车

关闭命令行，输入修改后的密码，即可进入系统

win10密码破解

点击开机，出现Windows图标重启（连续三次以上）

看到==正在准备自动修复==停止操作，等待电脑跳转到自动修复界面

在自动修复界面，点击==高级选项== –》==疑难答疑== –》==高级选项== –》==命令提示符== 调出cmd命令行

命令行输入notepad 命令调出记事本

点击左上角文件—》打开—》找到系统盘—》 Windows—》System32

文件类型选择==所有==文件

找到sethc 文件重命名为sethc-back

找到cmd文件 复制一份，将复制出来的重命名为sethc

关闭 笔记本和cmd命令行窗口，点击关闭电脑，并重新开机

开机到登录界面，按5次shift键位调出cmd命令行

输入net user zc 123.com

改完后即可用新密码登录win10系统
