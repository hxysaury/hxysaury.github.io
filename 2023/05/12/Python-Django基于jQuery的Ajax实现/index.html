AJAXпјҲAsynchronous Javascript And XMLпјүзҝ»иҜ‘жҲҗдёӯж–Үе°ұжҳҜвҖңејӮжӯҘзҡ„Javascriptе’ҢXMLвҖқгҖӮеҚідҪҝз”ЁJavascriptиҜӯиЁҖдёҺжңҚеҠЎеҷЁиҝӣиЎҢејӮжӯҘдәӨдә’пјҢдј иҫ“зҡ„ж•°жҚ®дёәXMLпјҲеҪ“з„¶пјҢдј иҫ“зҡ„ж•°жҚ®дёҚеҸӘжҳҜXMLпјүгҖӮ

AJAX дёҚжҳҜж–°зҡ„зј–зЁӢиҜӯиЁҖпјҢиҖҢжҳҜдёҖз§ҚдҪҝз”ЁзҺ°жңүж ҮеҮҶзҡ„ж–°ж–№жі•гҖӮ

==AJAX жңҖеӨ§зҡ„дјҳзӮ№жҳҜеңЁдёҚйҮҚж–°еҠ иҪҪж•ҙдёӘйЎөйқўзҡ„жғ…еҶөдёӢпјҢеҸҜд»ҘдёҺжңҚеҠЎеҷЁдәӨжҚўж•°жҚ®е№¶жӣҙж–°йғЁеҲҶзҪ‘йЎөеҶ…е®№гҖӮпјҲиҝҷдёҖзү№зӮ№з»ҷз”ЁжҲ·зҡ„ж„ҹеҸ—жҳҜеңЁдёҚзҹҘдёҚи§үдёӯе®ҢжҲҗиҜ·жұӮе’Ңе“Қеә”иҝҮзЁӢпјү==

AJAX дёҚйңҖиҰҒд»»дҪ•жөҸи§ҲеҷЁжҸ’д»¶пјҢдҪҶйңҖиҰҒз”ЁжҲ·е…Ғи®ёJavaScriptеңЁжөҸи§ҲеҷЁдёҠжү§иЎҢгҖӮ

  • еҗҢжӯҘдәӨдә’пјҡе®ўжҲ·з«ҜеҸ‘еҮәдёҖдёӘиҜ·жұӮеҗҺпјҢйңҖиҰҒзӯүеҫ…жңҚеҠЎеҷЁе“Қеә”з»“жқҹеҗҺпјҢжүҚиғҪеҸ‘еҮәз¬¬дәҢдёӘиҜ·жұӮпјӣ
  • ејӮжӯҘдәӨдә’пјҡе®ўжҲ·з«ҜеҸ‘еҮәдёҖдёӘиҜ·жұӮеҗҺпјҢж— йңҖзӯүеҫ…жңҚеҠЎеҷЁе“Қеә”з»“жқҹпјҢе°ұеҸҜд»ҘеҸ‘еҮәз¬¬дәҢдёӘиҜ·жұӮгҖӮ
