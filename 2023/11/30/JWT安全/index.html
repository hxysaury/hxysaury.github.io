cookie(ж”ҫеңЁжөҸи§ҲеҷЁ)

вҖӢ cookie жҳҜдёҖдёӘйқһеёёе…·дҪ“зҡ„дёңиҘҝпјҢжҢҮзҡ„е°ұжҳҜжөҸи§ҲеҷЁйҮҢйқўиғҪж°ёд№…еӯҳеӮЁзҡ„дёҖз§Қж•°жҚ®пјҢд»…д»…жҳҜжөҸ и§ҲеҷЁе®һзҺ°зҡ„дёҖз§Қж•°жҚ®еӯҳеӮЁеҠҹиғҪгҖӮ

cookie з”ұжңҚеҠЎеҷЁз”ҹжҲҗпјҢеҸ‘йҖҒз»ҷжөҸи§ҲеҷЁпјҢжөҸи§ҲеҷЁжҠҠ cookie д»Ҙ kv еҪўејҸдҝқеӯҳеҲ°жҹҗдёӘзӣ®еҪ•дёӢзҡ„ ж–Үжң¬ж–Үд»¶еҶ…пјҢдёӢдёҖж¬ЎиҜ·жұӮеҗҢдёҖзҪ‘з«ҷж—¶дјҡжҠҠиҜҘ cookie еҸ‘йҖҒз»ҷжңҚеҠЎеҷЁгҖӮз”ұдәҺ cookie жҳҜеӯҳеңЁе®ўжҲ· з«ҜдёҠзҡ„пјҢжүҖд»ҘжөҸи§ҲеҷЁеҠ е…ҘдәҶдёҖдәӣйҷҗеҲ¶зЎ®дҝқ cookie дёҚдјҡиў«жҒ¶ж„ҸдҪҝз”ЁпјҢеҗҢж—¶дёҚдјҡеҚ жҚ®еӨӘеӨҡзЈҒзӣҳз©ә й—ҙпјҢжүҖд»ҘжҜҸдёӘеҹҹзҡ„ cookie ж•°йҮҸжҳҜжңүйҷҗзҡ„

session(ж”ҫеңЁ жңҚеҠЎеҷЁ)

session д»Һеӯ—йқўдёҠи®ІпјҢе°ұжҳҜдјҡиҜқгҖӮиҝҷдёӘе°ұзұ»дјјдәҺдҪ е’ҢдёҖдёӘдәәдәӨи°ҲпјҢдҪ жҖҺд№ҲзҹҘйҒ“еҪ“еүҚе’ҢдҪ дәӨ и°Ҳзҡ„жҳҜеј дёүиҖҢдёҚжҳҜжқҺеӣӣе‘ўпјҹеҜ№ж–№иӮҜе®ҡжңүжҹҗз§Қзү№еҫҒпјҲй•ҝзӣёзӯүпјүиЎЁжҳҺд»–е°ұжҳҜеј дёүгҖӮ

session д№ҹжҳҜзұ»дјјзҡ„йҒ“зҗҶпјҢжңҚеҠЎеҷЁиҰҒзҹҘйҒ“еҪ“еүҚеҸ‘иҜ·жұӮз»ҷиҮӘе·ұзҡ„жҳҜи°ҒгҖӮдёәдәҶеҒҡиҝҷз§ҚеҢәеҲҶпјҢжңҚ еҠЎеҷЁе°ұиҰҒз»ҷжҜҸдёӘе®ўжҲ·з«ҜеҲҶй…ҚдёҚеҗҢзҡ„вҖңиә«д»Ҫж ҮиҜҶвҖқпјҢз„¶еҗҺе®ўжҲ·з«ҜжҜҸж¬Ўеҗ‘жңҚеҠЎеҷЁеҸ‘иҜ·жұӮзҡ„ж—¶еҖҷпјҢйғҪ еёҰдёҠиҝҷдёӘвҖңиә«д»Ҫж ҮиҜҶвҖқпјҢжңҚеҠЎеҷЁе°ұзҹҘйҒ“иҝҷдёӘиҜ·жұӮжқҘиҮӘдәҺи°ҒдәҶгҖӮиҮідәҺе®ўжҲ·з«ҜжҖҺд№ҲдҝқеӯҳиҝҷдёӘвҖңиә«д»Ҫ ж ҮиҜҶвҖқпјҢеҸҜд»ҘжңүеҫҲеӨҡз§Қж–№ејҸпјҢеҜ№дәҺжөҸи§ҲеҷЁе®ўжҲ·з«ҜпјҢеӨ§е®¶йғҪй»ҳи®ӨйҮҮз”Ё cookie зҡ„ж–№ејҸгҖӮ

жңҚеҠЎеҷЁдҪҝз”Ё session жҠҠз”ЁжҲ·зҡ„дҝЎжҒҜдёҙж—¶дҝқеӯҳеңЁдәҶжңҚеҠЎеҷЁдёҠпјҢз”ЁжҲ·зҰ»ејҖзҪ‘з«ҷеҗҺ session дјҡиў« й”ҖжҜҒгҖӮиҝҷз§Қз”ЁжҲ·дҝЎжҒҜеӯҳеӮЁж–№ејҸзӣёеҜ№ cookie жқҘиҜҙжӣҙе®үе…ЁпјҢеҸҜжҳҜ session жңүдёҖдёӘзјәйҷ·пјҡеҰӮжһң web жңҚеҠЎеҷЁеҒҡдәҶиҙҹиҪҪеқҮиЎЎпјҢйӮЈд№ҲдёӢдёҖдёӘж“ҚдҪңиҜ·жұӮеҲ°дәҶеҸҰдёҖеҸ°жңҚеҠЎеҷЁзҡ„ж—¶еҖҷ session дјҡдёўеӨұгҖӮ

token

TokenжҳҜз”ЁжҲ·иҝӣиЎҢдёҖдәӣжқғйҷҗж“ҚдҪңж—¶зҡ„и®ёеҸҜеҮӯиҜҒгҖӮtokenжң¬иҙЁжҳҜеӯ—з¬ҰдёІпјҢйҮҢйқўеҢ…еҗ«дәҶз”ЁжҲ·дҝЎжҒҜпјҢиҝҮжңҹж—¶й—ҙпјҢеҠ еҜҶж–№ејҸзӯүгҖӮtokenжҳҜеңЁеүҚз«ҜиҝӣиЎҢзҷ»еҪ•д№ӢеҗҺпјҢз”ұжңҚеҠЎеҷЁеҲҶеҸ‘з»ҷеүҚз«ҜпјҢз„¶еҗҺеүҚз«ҜиҝӣиЎҢжқғйҷҗж“ҚдҪңж—¶пјҢеҶҚе°ҶtokenеҸ‘йҖҒз»ҷжңҚеҠЎеҷЁпјҢз”ұжңҚеҠЎеҷЁжқҘйӘҢиҜҒгҖӮtokenжҳҜжңүиҝҮжңҹж—¶й—ҙзҡ„пјҢдёҖдҪҶtokenиҝҮжңҹпјҢз”ЁжҲ·е°ұиҰҒйҮҚж–°зҷ»йҷҶи®©жңҚеҠЎеҷЁз”ҹжҲҗж–°зҡ„tokenгҖӮ

еңЁ Web йўҶеҹҹеҹәдәҺ Token зҡ„иә«д»ҪйӘҢиҜҒйҡҸеӨ„еҸҜи§ҒгҖӮеңЁеӨ§еӨҡж•°дҪҝз”Ё Web API зҡ„дә’иҒ”зҪ‘е…¬еҸёдёӯпјҢ tokens жҳҜеӨҡз”ЁжҲ·дёӢеӨ„зҗҶи®ӨиҜҒзҡ„жңҖдҪіж–№ејҸгҖӮ

д»ҘдёӢеҮ зӮ№зү№жҖ§дјҡи®©дҪ еңЁзЁӢеәҸдёӯдҪҝз”ЁеҹәдәҺ Token зҡ„иә«д»ҪйӘҢиҜҒ

  1. ж— зҠ¶жҖҒгҖҒеҸҜжү©еұ•
  2. ж”ҜжҢҒз§»еҠЁи®ҫеӨҮ
  3. и·ЁзЁӢеәҸи°ғз”Ё
  4. е®үе…Ё

еӨҙйғЁйҖҡеёёз”ұдёӨйғЁеҲҶз»„жҲҗпјҡз®—жі•зұ»еһӢе’Ңд»ӨзүҢзұ»еһӢгҖӮ

  • з®—жі•зұ»еһӢпјҡжҢҮе®ҡз”ЁдәҺз”ҹжҲҗзӯҫеҗҚзҡ„з®—жі•пјҢдҫӢеҰӮ HMACгҖҒRSA жҲ–иҖ… ECDSAгҖӮ
  • д»ӨзүҢзұ»еһӢпјҡжҢҮе®ҡд»ӨзүҢзҡ„зұ»еһӢпјҢеёёи§Ғзҡ„жҳҜ JWTгҖӮ

еӨҙйғЁдҪҝз”Ё Base64URL зј–з ҒиЎЁзӨәпјҢе№¶дҪңдёәж•ҙдёӘ JWT зҡ„з¬¬дёҖйғЁеҲҶгҖӮеӨҙйғЁзҡ„дёҖдёӘзӨәдҫӢпјҡ

payload

иҪҪиҚ·еӯҳеӮЁдәҶжңүе…із”ЁжҲ·жҲ–е®һдҪ“зҡ„еЈ°жҳҺе’Ңе…¶д»–жңүе…ідҝЎжҒҜ

  • еЈ°жҳҺпјҡеҰӮз”ЁжҲ· IDгҖҒи§’иүІгҖҒжқғйҷҗзӯүдҝЎжҒҜ
  • жіЁеҶҢеЈ°жҳҺпјҡеҢ…еҗ«дёҖдәӣж ҮеҮҶзҡ„еЈ°жҳҺпјҲжҜ”еҰӮеҸ‘иЎҢдәәгҖҒиҝҮжңҹж—¶й—ҙзӯүпјүе’ҢдёҖдәӣиҮӘе®ҡд№үзҡ„ еЈ°жҳҺ

иҪҪиҚ·д№ҹдҪҝз”Ё Base64URLзј–з ҒиЎЁзӨәпјҢе№¶дҪңдёәж•ҙдёӘ JWT зҡ„з¬¬дәҢйғЁеҲҶгҖӮиҪҪиҚ·зҡ„дёҖдёӘзӨәдҫӢпјҡ

Signature

зӯҫеҗҚжҳҜжӢҝеҲ°иў«base64зј–з Ғзҡ„еӨҙйғЁе’ҢиҙҹиҪҪеҗҺпјҢз”Ёж ҮеӨҙйҮҢзҡ„еҠ еҜҶз®—жі•еҜ№жӢҝеҲ°зҡ„ж ҮеӨҙе’ҢиҙҹиҪҪиҝӣиЎҢеҠ еҜҶпјҢз„¶еҗҺдҪңдёәjwtзҡ„з¬¬дёүйғЁеҲҶгҖӮзӯҫеҗҚжҳҜз”ЁжқҘйӘҢиҜҒж ҮеӨҙе’ҢиҙҹиҪҪдёӯзҡ„дҝЎжҒҜжңүжІЎжңүиў«зҜЎж”№гҖӮеҰӮжһңж ҮеӨҙе’ҢиҙҹиҪҪзҡ„дҝЎжҒҜиў«зҜЎж”№пјҢеҲҷиҝҷдёӘjwtжҳҜеӨұж•Ҳзҡ„гҖӮеҸҚд№ӢйӘҢиҜҒйҖҡиҝҮ вҖ”вҖ”вҖ”вҖ”вҖ”-з”ЁдәҺйӘҢиҜҒ JWT зҡ„е®Ңж•ҙ жҖ§е’Ңзңҹе®һжҖ§

жңҚеҠЎеҷЁжңүдёҖдёӘдёҚдјҡеҸ‘йҖҒз»ҷе®ўжҲ·з«Ҝзҡ„еҜҶз ҒпјҲsecretпјүпјҢз”ЁеӨҙйғЁдёӯжҢҮе®ҡзҡ„з®—жі•еҜ№еӨҙйғЁе’ҢеЈ°жҳҺзҡ„еҶ…е®№з”Ё жӯӨеҜҶз ҒиҝӣиЎҢеҠ еҜҶпјҢз”ҹжҲҗзҡ„еӯ—з¬ҰдёІе°ұжҳҜ JWT зҡ„зӯҫеҗҚ

зӯҫеҗҚз”ҹжҲҗж–№ејҸпјҡе°ҶеӨҙйғЁе’ҢиҪҪиҚ·иҝӣиЎҢ Base64URL зј–з ҒеҗҺжӢјжҺҘеңЁдёҖиө·пјҢз„¶еҗҺдҪҝ з”ЁжҢҮе®ҡзҡ„еҠ еҜҶз®—жі•пјҲеҰӮ HMACгҖҒRSAпјүиҝӣиЎҢзӯҫеҗҚпјҢе°Ҷз”ҹжҲҗзҡ„зӯҫеҗҚж·»еҠ еҲ°JWTдёӯ
